【五】上下文无关语言

发表于 2024-11-11 更新于 2024-12-18

上下文无关语言的泵引理

回忆一下正则语言的泵引理(Pumping Lemma)：

如果这个正则语言被有限自动机识别，因为自动机的状态数是有限的，当字符串长度大于状态数时，必然有某些状态被循环重复。

正则语言的泵引理

对于每一个正则语言$ L$ 来说，存在一个整数$ n $，使得对于$ L $中的每一个长度$ \ge n $的字符串$ w $，我们可以写作$ w=xyz $，满足：

  1. $|xy|\leq n$
  2. $|y|>0$
  3. $\forall i\ge 0,xy^{i}z\in L$

对于上下文无关语言来说，我们可以找到两个部分，将它们任意泵进泵出。

上下文无关语言的泵引理

对于每一个上下文无关语言$ L $，存在一个整数$ n $，使得$\forall z\in L,|z|\ge n,\exists z=uvwxy$，满足：

  1. $|vwx|\leq n$
  2. $|vx|>0$
  3. $\forall i\ge 0,uv^iwx^iy\in L$

泵引理的证明

考虑$L-\left\{\epsilon\right\}$的CNF（乔姆斯基范式）一个文法，令该文法有$m$个不同的变量，取$n=2^{m}$。

取$z\in L \land |z|\ge n$，有引理1：产出$z$的解析树一定存在一条长度至少为$m+2$的路径。

由引理1，一定存在路径，有至少$m+1$个变量（去掉终结符）。由容斥原理，考虑一条最长的路径，在低$m+1$个非叶结点中，一定存在有两个结点，标记的是同一个变量，不妨记为$A$.

而$A$一定有两条候选的推导方式，$A\Rightarrow vAx$和$A\Rightarrow w$。

其中，$|vw|\not ={0}$，因为CNF文法中不含空产生式。

并且，$|vwx|\leq 2^{m}=n$，因为我们选取的是低$m+1$个非叶结点，树高最多为$m+1$。

引理1的证明

假设一个CNF文法的解析树的所有路径长度都小于等于$m+1$，则产生最长字符串长度为$2^{m-1}$，矛盾。

泵引理的使用

上下文无关语言的泵引理主要被用来证明某个语言不是上下文无关语言。
例如：

例1

$z=0^{n}10^{n}10^{n}$

令$z=uvwxy,|vwx|\leq n\land |vx|\ge 1$。

情况一： $vx$中没有 0

$vx$中一定有1，那么重复就会导致超过两个 1 ，矛盾。

情况二： $vx$中至少有一个 0

因为$|vwx|\leq n$，所以无法完整覆盖到所有的 0 串，所以$|uwy|$中会出现不足 n 个 0 的 0 串，矛盾。

上下文无关语言的判定性质

本届主要介绍上下文无关语言的一些性质如何去判定。

需要注意的是，有一些问题在正则语言中可以判定，但是在上下文无关语言中不能判定。比如判断两个上下文无关语言是否相同。

判断是否为空

等价于去除无用变量。

判断字符串是否属于语言

我们想判断字符串$\omega$是否属于上下文无关文法$G$描述的语言$L$。

情况一：$\omega=\epsilon$

只需要简单判断起始符号是否可以为空。

情况二：$\omega \not =\epsilon$
使用CYK算法，可以在时间复杂度$O(|\omega|^{3})$内判断。

CYK算法

令$\omega=a_1a_2\cdots a_n$，我们将构造一个$n\times n$的三角形变量矩阵

其中，$X_{ij}=\left\{A|A\Rightarrow a_i\cdots a_j\right\}$

进行归纳构造。
基础情况：$X_{ij}=\left\{A|A\rightarrow a_i\in P\right\}$

归纳：$X_{ij}=\left\{A|A\rightarrow BC \in P\right\}$

$B\in X_{ik}$

$C\in X_{k+1,j}$

最后只需要判断$S\in X_{1,n}$

判断是否无穷

使用泵引理常数$n$，如果该语言存在一个长度在$n$到$2n−1$之间的字符串，那么这个语言就是无穷的，否则就是有穷的。

上下文无关语言的闭包性质

并集下的闭包

令$L$和$M$是上下文无关文法G和H的上下文无关语言，只需要假设没有共同变量，如有则重命名。

$L\cup M$只需增加$S\rightarrow S_1|S_2$。

拼接下的闭包

令$L$和$M$是上下文无关文法G和H的上下文无关语言，只需要假设没有共同变量，如有则重命名。

$LM$只需增加$S\rightarrow S_1S_2$。

星闭包下的闭包

令$L$是上下文无关文法G的上下文无关语言。

$L^*$只需增加$S\rightarrow S_1S|\epsilon$。

翻转操作下的闭包

令$L$是上下文无关文法G的上下文无关语言。

只需要翻转每一个产生式体来构造$L^R$的文法。

同态操作下的闭包

令$L$是上下文无关文法G的上下文无关语言。

只需要将每个终结符$a$替换成$h(a)$来构造$h(L)$的文法。

交集和差集

上下文无关语言在交集操作下并不封闭。

上下文无关语言在差集操作下也不封闭。

证明差集操作和交集操作的等价性

$L\cap M=L-(L-M)$
所以任何在差集下封闭的语言，在交集下也封闭。

上下文无关语言和正则语言的交集是一个上下文无关语言

这个性质的证明过程包含了将一个DFA和一个PDA同时运行，它们组合起来依旧是PDA。这里PDA是根据终止状态接收的。

构造过程与乘积自动机类似但是多了一个栈。

逆同态下的闭包性质

可以通过构造新的PDA的方式来证明这个性质。
